== Installation ==


1. Go to plugins ->Add New-->Upload and select file `horizontal-post-slider.zip` and click upload.
1. Activate the plugin after upload completed.
1. Go to 'post slider options' in left menu then set your preference.
1. Place `[hori-post-slider]` in your wordpress page or post
